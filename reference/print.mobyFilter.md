# Print method for filterDetections() results

Prints a compact overview of a `mobyFilter` object: kept/removed totals
and the per-individual summary table. Inspect `$data`, `$data_discarded`
and `$summary` for details.

## Usage

``` r
# S3 method for class 'mobyFilter'
print(x, ...)
```

## Arguments

- x:

  A `mobyFilter` object.

- ...:

  Ignored.

## Value

`x`, invisibly.
